# Arena Royale

A feature-rich, Clash Royale-inspired browser game built entirely in a single HTML file. No installation required - just open and play!

## Play Now

Simply open `index.html` in any modern web browser to start playing.

## Features

### Core Gameplay
- **Real-time Battle System** - Deploy troops, cast spells, and destroy enemy towers
- **Elixir Management** - Strategic resource management with double elixir overtime
- **Trophy System** - Climb the ranks and unlock new arenas
- **Multiple Game Modes** - Normal, 2v2 Duo, Draft, Chaos, and Tournament modes

### Cards & Deck Building
- **60+ Unique Cards** - Troops, spells, and buildings across multiple rarities
- **Card Rarities** - Common, Rare, Epic, Legendary, Champion, and Evolution cards
- **Deck Slots** - Save and switch between multiple custom decks
- **Card Upgrades** - Level up cards with shards and gold

### Progression Systems
- **Trophy Road** - Unlock rewards as you climb the ladder
- **Battle Pass (Pass Royale)** - Free and Premium reward tracks with exclusive content
- **Daily Login Rewards** - Claim bonuses every day
- **Challenges & Achievements** - Complete objectives for extra rewards
- **Card Mastery** - Track mastery progress for each card

### Economy & Shop
- **Gold & Gems** - Dual currency system
- **Star Points Shop** - Spend star points on exclusive cosmetics
- **Daily Shop Offers** - Rotating deals on cards and items
- **Chest System** - Earn and open chests for cards and resources

### Special Features
- **Wild Card Workshop** - Forge, summon, boost, and transmute cards
- **Lucky Spin Wheel** - Daily free spins for bonus rewards
- **Treasure Hunt Mini-Game** - Find hidden treasures for prizes
- **Tournament System** - Compete in brackets for glory
- **Clan System** - Join clans and donate cards
- **Meta Tier List** - View current deck rankings
- **Battle Log** - Review your match history

### Customization
- **Tower Skins** - Personalize your tower appearance
- **Player Titles** - Earn and display achievement titles
- **Emotes** - Express yourself during battles

### Visual Effects
- **Victory Celebrations** - Confetti and loot drops on wins
- **Smooth Animations** - Polished UI transitions throughout
- **Troop Trails** - Visual effects for moving units
- **Dynamic Weather** - Seasonal themes with effects like snowfall

## Tech Stack

- Pure HTML5, CSS3, and Vanilla JavaScript
- No external dependencies or frameworks required
- Responsive design optimized for mobile and desktop
- Local storage for game save data

## Browser Support

Works on all modern browsers:

- Firefox
